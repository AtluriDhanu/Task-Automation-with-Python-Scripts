# Task Automation with Python Scripts
A basic Python automation script that moves all .jpg files from a source folder to a destination folder. The project demonstrates task automation using the os and shutil modules, file handling, loops, and user input to simplify repetitive file management tasks.

# Problem Statement
Automate a repetitive task by moving all .jpg image files from one folder to another using Python.

# Key Features
- Accepts source and destination folder paths from the user.
- Automatically detects all .jpg files.
- Moves files to the destination folder.
- Creates the destination folder if it does not exist.
- Displays the status of each moved file.

Concepts Used
- os Module
- shutil Module
- Requests
- File Handling
